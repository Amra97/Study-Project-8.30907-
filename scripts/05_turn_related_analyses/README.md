## What the folder conatins ?

## This folder contains the code for analysis based on when participants turned into a street. What objects were view, fixated on, dwelled on are in the respective sub-folders. The goal here is to see if when navigating as in the real world what affect landmarks, bodies, faces and other objects in the environment have on a persons eye movements and navigationally relevant behaviour.
## Below is a list of each script in the repositry and a very short description of what in intends to do. For more details please read the code description when viewing files separately. 

- `fixations_per_collider_turn1&2.py` — Computes fixation counts per collider or object around turns, including comparisons between first and repeated street entries.
