## Scripts

- `identify_turns_multiple_participants.ipynb` — Supports the identification and annotation of turn onsets and repeated street entries across participants. This script is part of the manual turn-labeling workflow used to define turn-related analysis windows and distinguish first from repeated entries into streets.

- `visualize_turns.ipynb` — Visualizes navigation trajectories together with the annotated turn points on the city map. It is mainly used for inspection and validation of the turn-labeling procedure and to ensure that annotated turns match the intended path segments.
