## Scripts

- `headvel_eyevel_fixationduration_objectcategory_glmm.Rmd` — Fits the mixed-effects models testing whether fixation duration and pre-fixation movement measures differ across object categories.

- `boxplots_headvel_eyevel_fixationduration.ipynb` — Produces descriptive visualizations of fixation duration, head velocity, and eye or saccade-related velocity measures across object categories.
